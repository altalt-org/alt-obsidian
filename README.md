# Alt

Free unlimited transcription and AI note-taking for Obsidian with the Alt desktop app

## Features

- Real-time transcription while recording
- Microphone and system audio capture
- AI-powered summaries, chat, and translation
- Markdown export

## Requirements

- [Alt desktop app](https://www.altalt.io) must be running
- Desktop only (macOS, Windows, Linux)

## Install

### Manual

1. Download `main.js`, `manifest.json`, and `styles.css` from a [release](https://github.com/altalt-org/alt-obsidian/releases)
2. Create `.obsidian/plugins/alt/` in your vault
3. Copy the three files into that folder
4. Reload Obsidian and enable **Alt** in Community Plugins

### BRAT

1. Install [BRAT](https://github.com/TfTHacker/obsidian42-brat) plugin
2. Add `altalt-org/alt-obsidian` as a beta plugin

## Usage

1. Open the recording panel from the ribbon icon or command palette
2. Start recording — transcriptions appear in real time
3. Use AI features (summary, translation, chat) from the panel tabs

## Notes

- System audio capture requires macOS Screen Recording permission
- Plugin language follows Obsidian's language setting
